esmb: The fastest way to run a shared ES6 module, from a bookmark!

Features

  1. JS editor, powered by CodeMirror
  2. ES6 static module imports
  3. No install
  4. No configuration
  5. Works on every page
  6. Works on every site
  7. Executing context control
  8. Customisable
  9. Safe
  10. Anonymous
  11. No referrer

Install

Simply drag the following link and drop it to the personal bar of your browser: esmb

Enjoy it!
